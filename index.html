HTML

    El HTML (Hypertext Markup Language) es el lenguaje de marcas (tags) estándar para la creación de páginas y aplicaciones en la Web.

    Junto con CSS (Cascading Style Sheets) y JavaScript, conforma el grupo de las tres tecnologías principales de la Web.

    Fue creado en 1990 por el científico británico Tim Berners-Lee. La primera página web está aún disponible en http://info.cern.ch/

HTML

    El HTML especifica la estructura y la semántica de una página web.

    La presentación es especificada mediante CSS y la interactividad mediante JavaScript.

    Tanto HTML como CSS son estándares del World Wide Web Consortium (W3C).

HTML5

    Su primera especificación fue publicada por el W3C en 2014.

    Reduce la necesidad de complementos externos (ej. Flash) para el manejo de multimedia y otros recursos. Para esto, se apoya en CSS, JavaScript y en el Document Object Model (DOM).

    Se diseñó para funcionar en diferentes tipos de dipositivos (ej. computadoras, tabletas, teléfonos).

El Document Object Model

Ejemplo de documento HTML

    
<!DOCTYPE html>
<html>
    <head>
        <title>Título del documento HTML</title>     
    </head>
    <body>
        Contenido del documento HTML
    </body>
</html>

Elementos de HTML

    Son los componentes individuales de un documento HTML. Contienen tags, pero no son lo mismo.

    Están compuestos de un tag inicial, un tag final (el cual opcional) y contenido. El tag final puede tener atributos especificados en la forma nombre=valor.
